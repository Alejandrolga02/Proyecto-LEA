Validador de numeros
Presione el boton de validar para continuar.

Resultado

  1. Obtiene la palabra introducida por el usuario
  2. Se convierte en un array con un carácter en cada índice
  3. Se obtiene el código ascii de cada carácter
  4. El código ascii se convierte en binario
  5. Se genera una tabla de transición a partir de un automata que valide numeros
  6. Se recorre cada bit de cada binario para validar la palabra
  7. Si algún caracter no es valido va a salir una alerta y si no saldrá una alerta diciendo que es correcto
